* API - набор операций, который можно выполнять с чем-либо:
  * Библиотека
  * Приложение
  * Сервис
  * Веб-сервис и т.д.
  * Класс
  * Интерфейс 
  * и т.д.
* Главный принцип разработки систем:
  * Принцип минимальной ответственности для каждого компонента системы
    * В этом случае любой компонент можно безболезненно заменить, расширить, масштабировать
* Бизнес-логика - алгоритмы системы, связанные с ее назначением
* Пример "денежный перевод"
  * - проверить баланс отправителя
  * - проверить его полномочия
  * - снять деньги с одного счета
  * - положить деньги на другой счет
  * - сделать уведомление о переводе
